Welcome to my workshop wrapped around the Twitter's API. To explore the possibilities of this workshop, you need to select a data set : you can create a query to dump tweets meeting your search criteria or simply explore the query database. Once your data set has been defined, you can sort the processed data with a powerful search engine, get basic information about them and visualize several kinds of graphics about them.
